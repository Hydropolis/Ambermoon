# Skills

There are 10 skills:
- Attack (ATT)
- Parry (PAR)
- Swim (SWI)
- Critical hit (CRI)
- Find traps (F-T)
- Disarm traps (D-T)
- Lockpicking (L-P)
- Searching (SRC)
- Read scrolls (R-S)
- Use magic (U-M)

The attribute values are stored as 4 uwords:

- uword[0]: Current value
- uword[1]: Max value
- uword[2]: Bonus from equipment etc
- uword[3]: Unknown

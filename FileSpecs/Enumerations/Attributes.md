# Attributes

There are 8 attributes:
- Strength (STR)
- Intelligence (INT)
- Dexterity (DEX)
- Speed (SPD)
- Constitution (CON)
- Charisma (CHA)
- Luck (LUK)
- Anti-Magic (A-M)

The attribute values are stored as 4 uwords:

- uword[0]: Current value
- uword[1]: Max value
- uword[2]: Bonus from equipment etc
- uword[3]: Unknown
